Quick Start

Cole os tokens CSS no seu projeto e comece a criar paginas com o visual do Web Designer do Futuro.

  1. Copie os Design Tokens acima para o seu CSS global
  2. Importe as fontes: @import url('...Inter...Syne...')
  3. Use font-family: var(--font-display) para headings
  4. Use font-family: var(--font-primary) para body
  5. Grid: display: grid; grid-template-columns: repeat(4, 1fr)
  6. Bordas: border: 1px solid var(--color-border)
  7. Selecao: ::selection { background: rgba(239,68,68,0.3); color: #fff }
